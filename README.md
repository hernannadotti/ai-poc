# AI POC - Azure DevOps Integration

AI Proof of Concept for IPG demonstrating AI agent integration with Azure DevOps.

## Description

This landing page showcases a POC for integrating intelligent AI agents with Azure DevOps to streamline development workflows, automate code reviews, and enhance team productivity.

## Tech Stack

- **React** (latest version) - UI framework
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS v3** - Utility-first CSS framework
- **Vite** - Fast build tool and dev server

## Design

Minimalistic design inspired by NYC signal and signage aesthetics:
- High-contrast black & white palette (subway signage style)
- Bold uppercase Helvetica typography
- Yellow caution-style info boxes (street sign inspired)
- White circular bullets (subway route indicators)
- Fully responsive layout

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173/`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
